# 微信小程序－天气预报

### 说明：

实现了天气预报和生活指数，设置城市等功能。

### 数据接口:

- https://api.getweapp.com/thirdparty/heweather/now
- https://api.getweapp.com/thirdparty/heweather/suggestion

### 目录结构：

- images — 存放项目图片 
- pages — 存放项目页面相关文件，包括about,city,index,logs,weather等页面
- utils — 存放日期格式化文件

### 开发环境：

微信web开发者工具 v0.10.102800
